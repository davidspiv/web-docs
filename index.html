HTML

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. It consists of a series of elements, which you use to enclose, or wrap, sections of content to make it appear or act a certain way.

Boilerplate HTML

  • <!DOCTYPE html>: Doctype is a historical artifact that needs to be included for everything else to work right. It is the shortest string of characters that counts as a valid doctype.
  • <html lang=“en”></html>: All the content on the page. It is sometimes known as the root element. Include the “lang” attribute to be indexed more effectively by search engines and useful to people with visual impairments.
  • <head></head>: Metadata about the document.
  • <meta charset="utf-8">: This element represents metadata that cannot be represented by other HTML meta-related elements, like <base>, <link>, <script>, <style> or <title>. The “charset” attribute specifies the character encoding of the document as UTF-8, which includes most characters from the vast majority of human written languages. With this setting, the page can now handle any textual content it might contain.
  • <title></title>: Title of the page, which is the title that appears in the browser tab the page is loaded in. The page title is also used to describe the page when it is bookmarked.
  • <body></body>: All the content for the document
